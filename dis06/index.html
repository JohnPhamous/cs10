Introduction

A function is a named block of code. Invoking a function name, knows as a function call causes the functions statements to execute.

A function definition consists of a new functions name and is sometimes followed by the preceding block of statements.

In the statement

void Print(string strName)

strName is a parameter. A function may have multiple parameters, seperated by commas. A function definiton without parameters must still have parenthasis. The value passed to a parameter is kmown as an agrument.

Each function must have a return statement which tranfers control of the program back to the where the functions was originally called.

The value of the return statement depends on how the function is defined. For example, in the function int main() a function named main is declared to have a type of int. If a function is declared to be void as it is in the example above, then the return statement should simply be return;

Designating blocks of codes as functions can be useful for three reasons:

1) Improving program readability
2) Modular program development
3) Avoiding redundant code
